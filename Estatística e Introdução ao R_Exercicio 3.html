Objetivo da anlise:

A base de dados escolhida foi a Melanoma do pacote MASS, a partir dela, o presente trabalho, busca analisar as variveis que podem ser mais influentes para levar o paciente a bito por melanoma. Para isso, ser ajustado um modelo de regresso logstica para prever a probabilidade de bito causado pelo melanoma.

Sobre a base de dados:

Para este exerccio foi escolhida a base de dados Melanoma do pacote MASS, esta base de dados contm informaes sobre 205 pacientes na dinamarca que possuiam melanoma maligno.

Sobre os dados, temos as seguintes variveis que sero utilizadas no exerccio:

status = bito pelo melanoma (1), vivo (2), bito por outras causas (3)
sex = Masculino (0), Feminino (1)
age = idade em anos
thickness = Espessura do tumor em mm
ulcer = Ulcerao do tumor (0 = no, 1 = sim)

A varivel resposta ser status e as variveis preditoras sero sex, age, thickness e ulcer.

O objetivo ajustar um modelo de regresso logstica para prever a probabilidade de bito causado pelo melanoma.

O intercepto negativo indica que, para pacientes com baixa espessura de tumor e sem lcera, as chances de sobrevivncia so consideravelmente mais altas.
