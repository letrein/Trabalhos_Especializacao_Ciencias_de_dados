Dataset: Dinosaour Genera

O Dataset escolhido provindo do Kaggle, chamado Dinosaour Genera, ele trs informaes sobre diversas espcies de dinossauros; o disponvel em: https://www.kaggle.com/datasets/canozensoy/dinosaur-genera-dataset.

A escolha do dataset se deu pela ideia de gerar um modelo de previso do tipo de dieta dos animais baseados nos atributos disponveis. Sendo eles: Dimenes fsicas, nvel de inteligncia, tipo de locomoo, informaes anatomicas; Ainda, o dataset tem informaes como nome cientifico e comum, local que habitava, era geolgicas, entre outros.

Desta forma, o objetivo deste trabalho gerar um modelo de classificao de dieta de dinossauros baseados em atributos selecionados.

import pyspark 

from pyspark.sql import SparkSession
from pyspark.ml import Pipeline
from pyspark.ml.feature import StringIndexer, VectorAssembler, OneHotEncoder
from pyspark.ml.classification import RandomForestClassifier
from pyspark.ml.evaluation import MulticlassClassificationEvaluator

#  Criar sesso Spark
spark = SparkSession.builder.appName("DinoDietClassifier").getOrCreate()


# Selecionar features relevantes
# Variveis numricas
numeric_features = ["length_m", "weight_kg", "height_m"]

# Variveis categricas a serem codificadas
categorical_features = ["locomotion", "geological_period"]

# Varivel alvo
label_column = "diet"

# Transformaes categricas
indexers = [StringIndexer(inputCol=col, outputCol=col + "_idx") for col in categorical_features]
encoder = OneHotEncoder(
    inputCols=[col + "_idx" for col in categorical_features],
    outputCols=[col + "_vec" for col in categorical_features]
)

# Indexador para o rtulo
label_indexer = StringIndexer(inputCol=label_column, outputCol="label")

# Vetorizao
assembler = VectorAssembler(
    inputCols=numeric_features + [col + "_vec" for col in categorical_features],
    outputCol="features"
)

# Modelo
rf = RandomForestClassifier(featuresCol="features", labelCol="label", numTrees=50)

# Pipeline completo
pipeline = Pipeline(stages=indexers + [encoder, label_indexer, assembler, rf])

# Separar em treino/teste
train_data, test_data = df.randomSplit([0.8, 0.2], seed=42)

# Treinar e avaliar
model = pipeline.fit(train_data)
predictions = model.transform(test_data)

evaluator = MulticlassClassificationEvaluator(labelCol="label", metricName="accuracy")
accuracy = evaluator.evaluate(predictions)
print("Acurcia do modelo com Spark MLlib:", accuracy)
